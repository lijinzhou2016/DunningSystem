# DunningSystem

### 说明
python bottle项目，管理订单信息 

### 环境
* windows or mac
* python 2.7.12
* mysql 5.7
### python依赖包
```
bottle==0.12.16
peewee==3.8.1
PyMySQL==0.9.3
PyYAML==3.13
xlrd==1.2.0
```


### 快速启动
* 配置config/conf.yml
* 创建数据库 dunning
* 执行 python db.py 生成数据表
* 手动录入数据库一个管理员账号
* 安装依赖 pip install -r request.txt
* 启动项目 python main.py
